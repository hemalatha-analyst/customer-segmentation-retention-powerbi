📊 Customer Segmentation & Retention Analytics in Power BI
📌 Project Overview

This project is a Power BI-based Customer Segmentation & Retention Analytics dashboard designed to analyze customer behavior, identify high-value segments, and track retention and churn patterns.

It helps businesses understand:

Which customers generate the most revenue
Which customers are at risk of leaving
How retention varies across segments, regions, and channels
How loyalty programs impact customer lifetime value

This project is built as a real-world business intelligence solution for CRM, marketing, and growth analytics teams.

🎯 Objective

To build an interactive Power BI dashboard that:

Segments customers based on behavior and spending
Analyzes customer retention and churn patterns
Identifies high-value and at-risk customers
Helps improve marketing and retention strategies
🧰 Tools & Technologies
Power BI Desktop
Power Query (Data Cleaning & Transformation)
DAX (Data Analysis Expressions)
Excel / CSV Dataset
Data Visualization
📁 Dataset Description

The dataset contains synthetic but realistic customer data with the following fields:

Customer ID
Customer Name
Age, Gender
City / Region
Occupation & Income Level
Join Date & Last Purchase Date
Purchase Frequency
Average Order Value
Total Spend
Product Category
Purchase Channel (Online / Offline / App)
Engagement Score
Satisfaction Score
Retention Status (Retained / At Risk / Churned)
Customer Segment (High / Medium / Low Value)
Loyalty Tier (Silver / Gold / Platinum)
⚙️ Power Query Transformations
Removed duplicate customer records
Handled missing/null values
Changed incorrect data types (dates, numbers)
Extracted Month & Year from dates
Created calculated columns (Revenue, CLV inputs)
Cleaned and standardized categorical fields
📐 Data Model
Customer Table (Dimension)
Transactions / Customer Activity Table (Fact)
Date Table (for time intelligence)

Relationships built using:

Customer ID
Date fields
📊 Key DAX Measures
Total Customers

Churn Rate %
